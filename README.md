# Chef Joharudin Portfolio

Static portfolio website for Chef Joharudin.

## Setup

1. Open the `index.html` file in your browser, or upload the project folder to your hosting provider.

2. If you want to use a local web server, use your preferred tool instead of `http-server`.

## Notes

- The project no longer includes the local `http-server` dependency.
- The `json` package is still declared in `package.json`.
- You can still use Node/npm for other workflow tools, but this site is a static HTML website that works by opening `index.html` directly.
